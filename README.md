## Getting Started

To checkout the project and build the site/symbols:

```bash
$ git clone git@github.com:spilth/national-park-symbols.git
$ cd national-park-symbols
$ bundle install
$ bundle exec middleman server
```

Then visit <http://localhost:4567>
